About

This web viewer demonstrates the ability to display processed frames from the Android Edge Detection application. In a production environment, frames would be received via WebSocket or HTTP endpoint.

Architecture: The Android app processes camera frames using OpenCV (C++) via JNI, renders them with OpenGL ES, and can send processed frames to this web viewer for display and analysis.
